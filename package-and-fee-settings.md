## 二、分类信息全局费用设置

费用设置存储在 `s_site_config` 表中，按功能模块分为以下几部分：

### 2.1 发布费用设置（PUBLISH_PAYMENT_SETTING）

> 控制用户发布分类信息时是否收费

| 字段 | 值 | 说明 |
|------|------|------|
| `publish_payment` | **2（免费）** | 1=收费，2=免费，3=关闭 |
| `publish_cost_setting` | `[{"cost":"2","days":"365"}]` | 若收费：¥2/条，有效365天 |
| `publish_payment_ios` | **2（免费）** | iOS端发布费用 |
| `publish_cost_setting_ios` | `[{"cost":"2","days":"365"}]` | iOS端收费设置 |

## 五、数据表关系总结

```
s_site_config (全局费用配置)
  ├── PUBLISH_PAYMENT_SETTING    → 发布费用全局设置
  ├── TOP_PAYMENT_SETTING        → 置顶费用全局设置
  ├── REFRESH_PAYMENT_SETTING    → 刷新费用全局设置
  ├── QY_PUBLISH_PAYMENT_SETTING → 企业发布费用设置
  ├── QY_TOP_PAYMENT_SETTING     → 企业置顶费用设置
  └── PAYMENT-SETTING            → 支付总开关

s_fenlei_category_perm (分类级费用覆盖)
  ├── follow_global=1 → 跟随 s_site_config 全局设置
  └── follow_global=0 → 使用分类自身费用配置

s_fenlei_package (套餐)
  ├── 用户购买套餐获得刷新/置顶/发布权益
  └── s_fenlei_package_user (用户已购套餐记录)
```
